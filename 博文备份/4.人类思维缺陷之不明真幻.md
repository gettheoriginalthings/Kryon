# 人类思维缺陷之不明真幻

**一切有为法如梦幻泡影，如露亦如电，亦如虚拟现实，应作如是观**

 

《四个果位》这篇文章里说了，要想彻底断掉尘世间的种种烦恼，最根本的方法就只有证悟空性。那证悟空性又是什么意思呢？简单的说就是：你明白了原来你以为很客观很真实的这个现实世界，其实是一个我们自己创造出来的“虚拟现实”，而我们人生经历的种种，都只是在这个虚拟现实游戏中的玩耍而已。明白了这点就算是了解了啥叫空性，等你能真的活出毫无烦恼的游戏人生境界时，就算是证悟了空性。

 

佛在说法的时候经常会用比喻，比如把一切有为法比作露珠和闪电。在两千多年前的时候也只有这些自然现象能够被用来比喻。但现在不一样了，我们有了电视、电影、电脑游戏、虚拟现实、增强现实。所以我们现代人有了这么多高科技的比喻之后，是应该能更容易理解和体会到佛说的“一切有为法如梦幻泡影”到底是什么意思。

 

大家都知道显示屏上的画面并不是连续的，而是一帧一帧的静态画面快速播放，让我们看起来好像觉得它是连续的动态。而一般显示器的帧率是60帧，也就是每秒播放60帧静态画面，这个速度已经能完全把我们的眼睛骗过去了。而现代科学已经发现，其实我们物理现实中的时间和空间也并不是连续的，而是量子化的，也就是离散的。而普朗克频率，也就是普朗克时间的倒数，大约是每秒1043,就是我们这个物理现实的帧率。这个话的意思就是，我们以为是连续的物理现实，其实是和我们所看的屏幕一样的，都是有帧率的，只是现实的帧率非常高，有每秒十的四十三次方，我们根本感觉不到，除非你有佛祖级别的定功。

 

另外这个时间帧率在不同的地方也是有不同的，也就是说时间流逝的快慢并不是个常量，而是可变的。这就是传说中的天上一天，人间一年。最近《西部世界》第三季的第二集也正好演绎了这一点。觉醒的接待员梅芙黑进虚拟现实服务器机房的监控录像时，发现录像中的机器人似乎动作缓慢，像是慢镜头，然后梅芙解释道，她所在的虚拟世界仿真速度比现实世界要快很多，也就是虚拟世界的时间帧率，或者说时钟频率要快很多，所以看现实世界的人都是慢动作。为啥是这样，大家可以自己琢磨琢磨。

 

《克里昂-人工智能，虚拟现实，多维人类》这篇信息中克里昂说在未来虚拟现实技术还能用来治病，我们进入虚拟现实医院中进行检查，治疗，还能在里面吃顿饭，然后出来病就好了。现在大多数人肯定觉得这不可能，呵呵。但如果你明白我们这个所谓的“真实”世界也是“虚拟”的之后，那就会明白这完全是可能的。而这其中的关键不在“虚拟”，而是在于这个世界不是我们以为的客观的，而是我们自己的信念创造出来的，在虚拟现实医院中只要我们的大脑认为身体的病好了，那它就同样能把健康在“现实”中创造出来。还记得《黑客帝国》中基努里维斯第一次参加训练时掉到了地上，从虚拟现实中出来时他发现自己流血了，然后他说：我以为这不是真的啊，怎么会流血呢？墨菲斯说：你的大脑认为它是真的，那它就是真的。的确就是这样的，这就是未来的虚拟现实医院能治病的原理，也说明了我们所谓的现实不是客观的，而是我们自己的信念创造的。

 

**如果一切都是幻，那什么才是真实？**

 

Only Love is real，唯有鄄攀钦媸档摹Ｕ饣笆敲淮淼模前文所说的那个永恒、无限的形而上的能量就是邸６我们平时所理解的劭赡苤皇且恢忠馐恫生的情绪现象，从没有把它和物理上真实的能量联系在一起。但实际上这个源头之鄄攀枪钩梢磺械淖罨本能量，是非常真实和物理的。克里昂曾经说过，在原子核和电子之间那个巨大的空间中所存在的就是这个神之邸

 

或许有人会问了，你前面不是说一切都是幻象吗？怎么现在又说这一切都是由这个真实的鄣哪芰抗钩傻模空饫镂侍獾墓丶就在于前面所说的我们对于真实的定义，我们把客观的东西定义为是真实的，而啥是客观的又是根据我们的感官来的，前五感能触碰的被我们定义为是客观的，单凭意识想象的是主观的。然后又认为客观的东西我们不能用主观意识直接去改变，所以就有了我们所认为的“客观真实”。而这种认识错误的关键就是，物质并不是客观的，不随我们的意识转移的，而是我们的意识信念共同投影，共同创造出来的。

 

所以认识一切是幻，不是意味着你要指着手机、指着苹果、指着一切东西然后对它们说这都是假的，这都是幻觉，这一切都没有意义。而是要明白它们都不是我们原来认为的那样“客观真实”。

 

净空老法师在讲经说，他认为真实的定义应该是：**能够永恒的东西才是真实的**。我觉得老法师的这个定义非常好。凼钦媸档模永恒的，但这个鄣哪芰克显化的一切物质现象却都会经历成住坏空这些阶段，没有一样是能够永恒的，永恒的只有它们的变化。所以转瞬即逝的物质现象不能算是真实的。

 

**执幻为真和借假修真**

 

我们在没觉悟的时候都是把物质幻象当作是真实，都在追求物质上的成功。觉醒了之后明白这一切都是幻，似乎又从中找不到人生的意义和该追求的东西。那讲具体一点，什么东西才是真的，才值得我们去追求呢？简单的说就是，**我们能带到下辈子去的东西都是真的**。那啥是能带走的呢？佛法里说了：万般将不去，唯有业随身！表害怕“业”这个字，它只是个中性词。物质身体虽然会衰变，但经验是能带的走的，宇宙是不会丢失信息的。有的人从小就对某些事情很有天赋，这就是上辈子带来的经验吗。克里昂还说过，觉悟也是能带走的，有个灵性法则就是，人们是不会退回到之前更加不觉悟的状态的。所以当你下辈子来的时候，总是能在你之前已经获得的觉悟层次上继续前进。

 

所以说我们要追求的，要修的，就是这些能带走的东西。说白了不就是经验值吗！所以说人生就是一场虚拟现实游戏吗！就跟打怪升级的角色扮演游戏很相似。所有的物质都是这场游戏里的道具，**所以物质也根本不需要是“真实”的，因为它们只是个道具吗，但也需要很“逼真”，这样才能让我们沉浸其中**。也就是说物质不是“客观真实”，而是我们自创的逼真道具。

 

**何为缘起性空**

 

人在有了足够的阅历之后都会明白，可能在刚开始的时候自己努力奋斗就是为了赚钱，但当成功了，有钱了之后，才发现自己享受的才是这个奋斗的过程。《深夜加油站遇见苏格拉底》这部电影里男主也说了类似的感悟。玩虚拟现实游戏的时候，你明知道画面是假的，但也无法阻止它带给你真实的感受，如果你去体验一把VR过山车，就知道我在说啥了。也就是说真实的画面和虚拟的画面给大脑带来的感受都是一样的，那或许你就会问了，那我们的大脑到底是怎么分辨真实和幻象的呢？**其实我们感受到物理世界的真实主要是靠触觉和体感，也就是触碰真实！**《盗梦空间》里说在梦境里死亡是无所谓的，因为死了只是从这层梦境里醒过来了，但在梦境里被打了一枪，痛苦却是真实的。

 

众生这个词的意思是：众缘和合而生。而缘的意思可以理解为各种关系、关联。众生的意思就是，万事万物都是有诸多因缘，也就是关系聚合而成的。我们一般观念都认为，是先有物质，东西，人，才有它们之间的关系，才有种种的体验。但实际情况则是，只有关系，体验才是真实的，物质只是作为道具的逼真幻象，也就是空。这就是缘起性空的大概意思。

**灵性生存**

 

灵性生存是克里昂提出的概念，意思就是我们人活在世上的主要任务是要获得灵性上的成长，意思层次上的提升。那如果我们的人生偏离了这个目的，就有可能会被灵魂拉下线，也就是挂掉。这就好比你玩角色扮演游戏时，你的那个角色有那么一点自由意志，结果你指望他去打怪升级，将来要当大统领，结果他却成天跑去花天酒地。于是你觉得这不行，就让他下线了，回到开始，重新设定角色参数再来一遍。

 

《机器人瓦力》这部动画片中未来飞船上的人个个都被机器人喂得像猪一样，或许我们会想，等将来人工智能技术突破了，我们是不是也能过上这样被机器人伺候的日子。或许是可以被伺候的，但我们是没法成天像猪那样娱乐的，因为我们得获得灵性上的成长，不然天天吃喝玩乐是会被叫下线的。而将来人们不需要做简单重复劳动之后，是会有科学、艺术、探索宇宙等很多事情可以去玩的。所以如果你有小孩的话，那可得培养他对科学艺术和宇宙的兴趣。不能眼界太低指望他学个手艺混口饭吃就行了，说不定到时就被人工智能取代了。

 

好好学习各种知识学问，天天向上提升我们的意识层次，这就是我们该做的事情。
